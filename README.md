# Ivy Dunlap - Professional Portfolio

A modern, responsive portfolio website showcasing Ivy Dunlap's organizational skills, AI aptitude, and project management expertise.

## Features

- **Modern Design**: Clean, professional layout with gradient accents and smooth animations
- **Responsive**: Fully responsive design that works on desktop, tablet, and mobile devices
- **Performance**: Lightweight, fast-loading site optimized for all devices
- **User-Focused**: Clear sections highlighting skills, projects, and contact information
- **Smooth Interactions**: Scroll animations, hover effects, and smooth navigation

## Sections

1. **Navigation Bar**: Sticky navigation with smooth scrolling to sections
2. **Hero Section**: Eye-catching introduction with call-to-action buttons
3. **About**: Overview of background, skills, and professional highlights with statistics
4. **Skills**: Six core competency cards highlighting:
   - Project Management
   - AI & Machine Learning
   - Organization & Planning
   - Technical Skills
   - Leadership & Communication
   - Strategic Thinking

5. **Projects**: Featured portfolio projects with descriptions and technology tags:
   - AI-Powered Task Orchestration System
   - Multi-Team Event Coordination Platform
   - Data Analytics & Reporting Dashboard
   - Collaborative Project Management Tool
   - AI Content Organization System
   - Smart Resource Allocation Engine

6. **Contact**: Contact information links and contact form

## Files

- `index.html` - Main HTML structure
- `styles.css` - Complete styling and responsive design
- `script.js` - Interactive features and animations

## Getting Started

1. Open `index.html` in your web browser
2. Customize the content with your actual projects and information
3. Update contact links (email, LinkedIn, GitHub)
4. Deploy to your hosting service

## Customization

### Update Contact Information
Replace `ivy@example.com` and social media links in:
- `index.html` - Contact section and links
- `script.js` - Contact form submission

### Modify Projects
Edit the project cards in the "Featured Projects" section with your actual:
- Project titles
- Descriptions
- Technology tags
- Project links (if available)

### Adjust Colors
Update the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent: #f5576c;
    ...
}
```

### Add More Content
- Add more skill cards in the Skills section
- Expand project cards with links
- Update About section with personal achievements
